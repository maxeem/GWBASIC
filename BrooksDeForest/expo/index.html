Tandy 1000 Game Title : Expo

System : Tandy(r) 1000EX / HX, 16 colors @ 160x200 resolution*
Year : 1990
Type : top-down RPG
Copyright : 1990 Brooks deForest (public domain released 2006)
*requires Microsoft(r) Tandy Basic 2.x


Expo is a top-down RPG game that has you walking around mazes, collecting keys and finding the exit before you run out of moves (or 'rounds').  To move use the keypad with the Num Lock on.
