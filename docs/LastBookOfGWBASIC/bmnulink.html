A Listing of the 17+ line

Ring Menu System

using

GW-BASIC

Note:  The ascii character (|=124) is on either side of a
function key at the bottom of each menu.  That represents a
substitute for the high order ascii box drawing characters
which surround the function keys on the original GW-BASIC
program menus.  They are 218, 196, 191, 179, 217, 192.

The HTML parser chokes on high order characters and won't
let me upload any file that has high order ascii characters
in the file.  Putting the boxes back around the function key
selections isn't that all of a problem.  Just do one top and
bottom line and then copy it to each menu changing the
line number if necessary.  For the char 179 you can just
search and replace alt 179 with alt 124 on the keypad.
For laptops, usually F11 toggles between keypad and text.
You stumble around a bit at first, but after a short time
you are off and running like a pro.  i didn't wanna upload
this file using a binary switch, even if that were possible.
  sorry...
